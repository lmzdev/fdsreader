# fdsreader
Ein Browserbasiertes Lesewerkzeug für Drägerware/FeuerON FDS-Dateien.

## Installation
```shell
# nodejs/npm required
npm install
npm run build
npm run preview
```

## Kompatibilität

Getestet mit FDS-Dateien folgender Leitstellensoftware:

- ISE Cobra
